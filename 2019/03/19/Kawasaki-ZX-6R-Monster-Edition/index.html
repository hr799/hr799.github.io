我对Kawasaki Ninja 250,Ninja 400和ZX-6R的看法

发表于 2019-03-19

从250到400到600, 可以看出我是个川崎粉, 趁着这次领回zx6r的机会, 来谈一下我对骑过的这几辆车粗浅的个人看法.

Kawasaki ninja 250 2012

典型的适合新手入门的型号, 排量小价格便宜, 中规中矩没有太多的电控, 确实适合练习基本功

优点
  1. 坐高友好, 适合身高不够的骑士, 女骑士
  2. 油耗低, 17L的油箱, 加满一次基本可以骑350km左右
缺点
  1. 车身前后配重不够均衡, 导致250在过弯的过程中不够自然, 会有头重脚轻的感觉
  2. 车身相对排量来说相对重, 180KG左右的干重
  3. 排量偏小, 在城市街道骑行堪用, 但是一旦上了高速, 要达到100km/h还是有点吃力 的, 基本要拧到9000-10000转

Kawasaki ninja 400 2018

2018年川崎全新推出的入门车款, 将忍者的排量从296cc提到了399cc, 一下子成为了这个排量级别中的王者, 超越了Yamaha R3

优点
  1. 车身轻巧, 168KG的干重, 比忍者300还要轻了6KG
  2. 马力提升, 马力从之前的39hp提升到了45hp, 再加上车身轻量化, 所以动力方面提升 了不少, 提速很快
  3. 车身前后配重1:1, 川崎这次是把H2家族的车身钢架下放到了入门车款, 这款车的配重真的没话说, 过弯非常轻松
  4. 配备滑动式离合器, 新手友好的东西, 在降档补油中可以有更高的容错不至于后轮锁死
  5. 配备档位显示, 400加入了档位显示, 对新手来说很好用, 不过骑习惯了一般都是通过听引擎声音来判断档位
缺点
  1. 油箱略小, 只有14L, 不过加满也能跑到350KM
  2. 车把略高, 因为400是街道骑行的设定, 所以出厂的时候手把略高, 如果能降低的话, 过弯压弯将会更自然
  3. 车fairing宽, 尾巴窄. 导流罩宽所以前脸看起来非常爆满有战斗感, 但是如果换个角度从车尾看, 就会觉得车尾略为单薄.

Kawasaki ZX-6R Monster Edition 2009

川崎家族中600级距的当家车款, 2007-2012之间为599cc, 从2013年起川崎又重新放出636cc的针对街道的zx6r. 我入的这款是2009年的Monster Edition限量款, 599cc.

优点
  1. 骑行姿势战斗, 说实话这个是我入手zx6r的很大一个原因, 他的姿势更街道跑车, 压弯更加舒服自然.
  2. 引擎声浪迷人, 四缸的引擎加上two Brothers尾段排气, 确实比双缸听起来凶猛了不少, 同时在路上能更有辨识度提升安全性.
  3. 马力强, 599cc的赛道车, 基本在1档就可以拧到130km/h的速度, 可见其有多暴力
  4. 路税低, 新西兰的路税只要在61cc - 600cc之间的, 路税为$405.71一年, 超过600cc的话, 路税将提升到$519.87每年.
缺点
  1. 发热大, 因为是高转发动机(最高可以到达20000转), 所以夏天骑行如果碰到红绿灯多, 温度很容易破100°导致风扇介入主动散热, 不过一旦骑起来, 因为有前置进风散热的关系, 温度会很快的降下来.
  2. 没有油表, 因为赛道车的设定, 所以这款zx6r是没有油量显示的, 仅仅会在油快要用完的时候亮灯提示, 日常骑行中需要骑士自己靠经验来判断油量
  3. 没有mode可调, zx6r在2013之后的款中才开始加入可调mode, 在sport, eco和雨天三个模式中调整, 所以没有可调mode也是一个小遗憾.

总结

相对于骑行的舒适度来说, ninja 400应该还是最舒服最适合日常通勤的
想要战斗骑姿, 热血跑山, 那么zx6r的暴力值得一试
250的话, 除非预算不足, 否则我会推荐新手直接入400,在各方面都相对250有不少的提升
