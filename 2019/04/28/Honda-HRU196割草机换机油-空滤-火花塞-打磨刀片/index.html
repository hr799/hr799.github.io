Honda HRU196割草机换机油+空滤+火花塞+打磨刀片

Posted on 2019-04-28 | In DIY
  • 作者：何容
  • 居住地：奥克兰
  • 发布时间：2019年4月27日

前言

新西兰几乎家家户户都有割草机, 有电力割草机也有汽油割草机, 那么对于使用汽油割草机的人来说, 定期保养是必不可少的.

家里的割草机是Honda HRU196, 购于2016年, 连续使用了接近3年, 最近感觉到已经不如之前那样顺手, 是时候给它做一次彻底的保养了.

保养项目

汽油割草机的核心是那颗内燃机, 内部有曲轴和燃烧室, 使用机油润滑保护, 火花塞点火, 所以对应的几个零部件属于消耗品, 该换就换

  1. 更换机油(10W30)
  2. 更换火花塞(型号BPR5ES)
  3. 打磨刀片
  4. 清理空滤
  5. 清洁机身

购买配件

本田的割草机, 官方手册推荐使用Honda原厂油, 质量最好. 不过最小起售是5升…对于一台割草机来说, 足够更换8次机油了. 所以这次我从Stihl Shop购买了科勒的946毫升装机油, 价格为18刀
火花塞一样, 不要买错型号, 购入价格13刀(实际supercheap只要大概9刀就可以买到)
空滤的话, 视具体情况再决定是否需要更换, 我检查了一下我的空滤挺干净, 所以这次不准备更换, 如果购买的话, 价格在25刀左右
刀片也一样, 购买全新一套是40刀(4刀片装). 这次我打算用Grinder打磨翻新一下继续使用

准备工具

  1. 接油盘
  2. 螺丝刀
  3. 扭力扳手
  4. 手套
  5. WD40
  6. 清洁布

    保养过程

看看用了3年的割草机, 上面满是灰尘和杂草

首先断开汽油开关, 把旋钮逆时针旋转90度

接着断开火花塞连接, 避免在过程中发动机误启动

拧开空滤盒子上方的2个螺母, 取出空滤芯体, 拆下芯体周围一圈的海绵, 然后内芯轻轻的在地上敲打出内部的灰尘, 不要试图用刷子刷灰尘, 那样反而会导致灰尘黏在过滤网上, 海绵放肥皂水里稍微洗洗干净, 晾干, 然后放在干净的机油里浸泡一会. 再拧掉多余的油之后, 就可以装回去了

用火花塞扳手拧下火花塞, 同理装上新火花塞,这步就不上图了
接着来更换机油, 拧开机油的盖子, 地上放好接机油的盘子, 然后把割草机向右侧翻(切记不要往左, 会导致汽油灌入空滤), 让机油自然流出, 换下来的脏机油不要随便扔, 回头装一个瓶子里, 送到附近的修车厂回收.

让机油慢慢流的同时我们可以开始拆刀片了, 使用扳手拆开固定地盘的螺丝

然后拆掉上面的4个刀片, 记住垫片的位置不要弄混了, 否则后期有可能会导致使用过程中螺丝松动刀片飞出的危险情况

可以看到3年的使用下来, 刀片已经有不少的伤了, 所以平时割草之前, 最好先检查一下草地是是否藏有石头或者其他硬物

拿出Grinder, 开始打磨刀片

4个刀片都打磨好之后, 看看效果

按照顺序依次装回刀片和底盘, 这里要非常注意, 底盘的螺丝, 要用大概49牛米 - 58.8牛米左右的力度拧紧, 刀片固定螺丝则是用29.4牛米-39.2牛米的力度拧紧. 太松的话, 可能会导致刀片松动, 太紧又有可能损伤螺丝, 产生隐患.
这里使用的是Toolpro的扭力扳手,推荐

再回到机油这边, 这时候旧机油也差不多滴干净了, 摆正割草机, 倒入大概600ml左右的新机油

衡量机油量是否足够的标志就是插入机油盖, 然后拿出来看看机油的最高线到哪, 如果是在头部1.5cm范围内的话, 就是正常的

使用WD40在割草机的4个轮子的轴承处喷一喷, 去锈又润滑, 到此为止, 割草机的保养就差不多了, 最后再使用水和布彻底清洁一下机身和底盘就好啦. 保养得当的本田割草机, 可以使用20年以上
