Honda HRU196割草机换机油+空滤+火花塞+打磨刀片

发表于 2019-04-28 | 分类于 DIY
  • 作者：何容
  • 居住地：奥克兰
  • 发布时间：2019年4月27日

前言

新西兰几乎家家户户都有割草机, 有电力割草机也有汽油割草机, 那么对于使用汽油割草机的人来说, 定期保养割草机是必不可少的.

家里的割草机是Honda HRU196, 购于2016年, 连续使用了接近3年, 最近感觉到已经不如之前那样顺手, 是时候给它做一次彻底的保养了.

保养项目

汽油割草机的核心是那颗内燃机, 内部有曲轴和燃烧室, 使用机油润滑保护, 火花塞点火, 所以对应的几个零部件属于消耗品, 该换就换

  1. 更换机油(10W30)
  2. 更换火花塞(型号BPR5ES)
  3. 打磨刀片
  4. 清理空滤
  5. 清洁机身

购买配件

本田的割草机, 官方手册推荐使用Honda原厂油, 质量最好. 不过最小起售是5升…对于一台割草机来说, 足够更换8次机油了. 所以这次我从Stihl Shop购买了科勒的946毫升装机油, 价格为18刀
火花塞一样, 不要买错型号, 购入价格13刀(实际supercheap只要大概9刀就可以买到)
空滤的话, 视具体情况再决定是否需要更换, 我检查了一下我的空滤挺干净, 所以这次不准备更换, 如果购买的话, 价格在25刀左右
刀片也一样, 购买全新一套是40刀(4刀片装). 这次我打算用Grinder打磨翻新一下继续使用

准备工具

  1. 接油盘
  2. 螺丝刀
  3. 扭力扳手
  4. 手套
  5. WD40
  6. 清洁布

新西兰智能浴霸排气扇安装DIY

发表于 2019-04-27 | 分类于 DIY
  • 作者：何容
  • 居住地：奥克兰
  • 发布时间：2019年4月27日

前言

本篇图文记录的是我为自家浴室安装智能风暖浴霸排气扇的全过程以及经验, 备份留档供日后排查问题使用.

简介

房子是在1970年代建造的, 后期洋人前屋主加建翻新过, 不过二楼的浴室没有预装排气扇, 导致洗完澡后水蒸气无法快速排出, 堆积在室内以致潮湿.
细菌在潮湿的环境下会迅速繁殖, 对健, 康产生隐患.
再者, 因为浴室没有浴霸, 冬天洗澡都是一种挑战。 所以，安装浴霸+排气扇的念头就出来了。

确定型号

浴霸有两种, 灯暖和风暖, 灯暖热得迅速但是范围小,再考虑到接下来准备要宝宝, 宝宝面朝上洗澡的时候, 灯暖对宝宝眼睛的伤害是巨大的, 所以确定风暖, 新西兰本地销售的都是灯暖机, 询问了不少大小商家都没有风暖型号.无奈之下只能考虑从中国购买.
国内的高端浴霸市场基本被日本松下霸占, 不过当时正巧小米旗下的yeelight开始众筹一款yeelight智能浴霸, 599人民币的价格, 支持风暖 + 照明 + 换气 + 干燥 + 吹风 + 无线遥控 + 手机App远程开关.
研究了一下参数之后, 果断拿下

配件

因为这款yeelight浴霸为300mm 600mm长宽, 设计给中国的集成吊顶使用的, 而新西兰又全都是非集成吊顶(基本全部为Gib板吊顶), 所以想要把这台机器安装在新西兰, 需要额外购置一个300 600的金属转接环, 淘宝50人民币拿下
传统的浴霸, 需要额外布线, 而这款浴霸, 采用了傻瓜式的走线方式, 只需要接火线+零线就可以了, 可以说大大提升了安装的方便性.

安装

先上一张安装前的图, 浴室为一个吸顶灯, 没有任何排气取暖设备

我对Kawasaki Ninja 250,Ninja 400和ZX-6R的看法

发表于 2019-03-19

从250到400到600, 可以看出我是个川崎粉, 趁着这次领回zx6r的机会, 来谈一下我对骑过的这几辆车粗浅的个人看法.

Kawasaki ninja 250 2012

典型的适合新手入门的型号, 排量小价格便宜, 中规中矩没有太多的电控, 确实适合练习基本功

优点
  1. 坐高友好, 适合身高不够的骑士, 女骑士
  2. 油耗低, 17L的油箱, 加满一次基本可以骑350km左右
缺点
  1. 车身前后配重不够均衡, 导致250在过弯的过程中不够自然, 会有头重脚轻的感觉
  2. 车身相对排量来说相对重, 180KG左右的干重
  3. 排量偏小, 在城市街道骑行堪用, 但是一旦上了高速, 要达到100km/h还是有点吃力 的, 基本要拧到9000-10000转

Kawasaki ninja 400 2018

2018年川崎全新推出的入门车款, 将忍者的排量从296cc提到了399cc, 一下子成为了这个排量级别中的王者, 超越了Yamaha R3
